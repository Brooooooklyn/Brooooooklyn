I build high-performance developer tools at the intersection of **Rust** and **JavaScript**. Creator of <a href="https://napi.rs" target="_blank">NAPI-RS</a> — the framework that powers Rust-based Node.js addons in <a href="https://rolldown.rs" target="_blank"><b>Rolldown</b></a>, <a href="https://oxc.rs" target="_blank"><b>Oxc</b></a>, <a href="https://cursor.com" target="_blank"><b>Cursor</b></a>, <a href="https://nextjs.org" target="_blank"><b>Next.js</b></a>, and hundreds of other projects.

Core team member of <a href="https://github.com/rolldown/rolldown" target="_blank">Rolldown</a>, <a href="https://github.com/oxc-project/oxc" target="_blank">OXC</a>, and contributor to <a href="https://github.com/swc-project/swc" target="_blank">SWC</a>.

- <a href="https://github.com/oxc-project/oxc" target="_blank"><b>OXC</b></a> — Team member. High-performance JS/TS toolchain (parser, linter, formatter, minifier) written in Rust.
- <a href="https://github.com/swc-project/swc" target="_blank"><b>SWC</b></a> — Built the Emotion CSS-in-JS transform plugin, maintained `swc-node` integration layer.
